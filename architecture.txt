Схема архитектуры NLP Analyzer

1. Основные компоненты:

   * Веб-сервер (main.go)
     - Обрабатывает HTTP-запросы
     - Маршрутизация (gorilla/mux)
     - Обслуживает статические файлы
   
   * AI модуль (ai/ai.go)
     - Классификация вопросов
     - Генерация ответов
     - Взаимодействие с базой знаний

   * База данных (db/db.go)
     - PostgreSQL
     - Миграции
     - Управление подключениями

   * Модели данных (models/models.go)
     - Структуры данных
     - Бизнес-логика

2. Основные потоки данных:
   Клиент -> Веб-сервер -> AI модуль -> База данных
                                     -> Ответ -> Клиент

3. Ключевые особенности:
   - Модульная архитектура
   - RESTful API
   - Шаблонизация (templates)
   - Конфигурация через переменные окружения
   - Система миграций базы данных

4. Технологический стек:
   - Go (backend)
   - PostgreSQL (база данных)
   - HTML/Templates (фронтенд)
   - Gorilla Mux (маршрутизация)
